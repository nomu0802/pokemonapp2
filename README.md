# pokemonapp2
育成したポケモンデータを登録することができるandroidアプリです。  
  
ポケモンのデータはポケモン徹底攻略様(https://yakkun.com/)　よりお借り致しました。  
画像データはhttps://github.com/PokeAPI/sprites様よりお借り致しました。  
趣味としての範囲で作成したアプリですが問題があれば早急に削除致します
